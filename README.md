# Multiverse Grammar Pro: Adaptive English Learning Engine

An interactive, gamified English learning platform designed to help students master grammar through a curated database of **5,245 hand-picked questions**. Built with a focus on pedagogical effectiveness and modern UI/UX.

![Version](https://img.shields.io/badge/version-3.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Tech](https://img.shields.io/badge/tech-Vanilla%20JS%20%7C%20Tailwind%20CSS-orange)

## 🚀 Key Features

*   **Curated Content:** Over 5,000 hand-picked questions across multiple grammar topics (Articles, Conditionals, etc.).
*   **Smart-Shuffle Algorithm:** Engineered a zero-repetition experience using `localStorage` to track answered questions across multiple sessions.
*   **Gamified Difficulty:** Three distinct modes (Easy, Medium, Hard) featuring a Lives system and timed challenges.
*   **Real-time Analytics:** Grayscale performance tracking that provides granular feedback on specific grammatical strengths and weaknesses.
*   **Pedagogical Feedback:** Instant corrective feedback with detailed explanations for every single question.

## 🛠️ Tech Stack

*   **Frontend:** Vanilla JavaScript (ES6+), HTML5
*   **Styling:** Tailwind CSS (Custom Comic-Book Theme)
*   **Persistence:** LocalStorage API
*   **Design:** Responsive Mobile-First Architecture

## 📖 How to Use

1.  **Select Topics:** Choose the specific grammar areas you want to practice.
2.  **Choose Difficulty:** 
    *   **Easy:** Infinite lives, no timer.
    *   **Medium:** 3 lives, no timer.
    *   **Hard:** 3 lives + 5-minute countdown.
3.  **Review:** After each game, use the **Topic Analysis** dashboard to see where you need more practice.

## 💻 Installation (Local)

Simply clone the repository and open `index.html` in any modern web browser.

```bash
git clone https://github.com/yourusername/multiverse-grammar-pro.git
cd multiverse-grammar-pro
# Open index.html in your browser
```
